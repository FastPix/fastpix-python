# AiResponseRecord

Represents an AI response record containing status and data for AI-generated features like summary, chapters, named entities, or moderation.

| ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| `status`                                                                                                      | *OptionalNullable[str]*                                                                                       | :heavy_minus_sign:                                                                                            | The status of the AI processing (for example, "available", "preparing", "failed").                            | ready                                                                                                         |
| `data`                                                                                                        | Dict[str, *Any*]                                                                                              | :heavy_minus_sign:                                                                                            | The AI-generated content data.
